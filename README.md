# Amazon-Sales-Customer-Analytics-Dashboard
This project provides a complete analytical breakdown of Amazon sales using interactive visual dashboards. The analysis highlights product performance, customer behavior, revenue distribution, and pricing trends across the United States.

Key Insights

Total Orders Analyzed: 60,019

Average Customer Rating: 3.0

Average Order Value (AOV): ₹2,545

Highest revenue generated through Credit Card, Net Banking & COD (~₹16.9M each)

Category comparison across Electronics, Clothing, Sports, Home & Books

State-wise price and sales distribution with geographical visualization.

Outcome & Business Value

This dashboard provides a data-driven way to:

Identify top revenue-driving product categories

Understand regional performance for sales expansion

Track customer sentiment to improve product quality

Improve operational decisions using payment & pricing insights
